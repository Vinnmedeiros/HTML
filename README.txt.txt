***Tags Importantes***
<h1> -> titulo 1
<h2> -> titulo 2, vai até h6
<p> -> Parágrafo
<br> -> Quebra Linha (Break Row)
<b> -> (Negrito Não semântico)
<strong> -> (negrito Semântico)
<i> - itálico (não semântico)
<em> -> itálico semântico (Emnfasis)
<mark> -> marcador (em amarelo)
<!-- --!> -> comentario
<big> -> texto grande
<small> -> texto pequeno
<ins> para sublinhado
<u> também para sublinhado
<del> para texto riscado, deletado.
<sup> do tipo x20 + 3 (x elevado a 20)
<sub> -> H2O (2 pequeno em baixo)
<code> -> Para mostrar o codigo como texto
<pre> -> Para identar o codigo como texto
<q> -> aspas entre o texto
<blockquote Cite" "=> -> Citação
<abbr> -> Abreviação, significado da palavra no site
<bdo dir="rtl"> -> texto invertido
<ol> -> Ordened List 
<ul> -> Unordened List
<li> -> Lista (ITENS)
<dl> -> Lista de definição
<dt> -> Texto
<dd> -> Descrição
<a href="http://" target="_blank" rel="external"> -> Link para site (abrindo outro link sem fechar o seu site, necessario blank e rel)
